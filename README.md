# Local Supermarket Management System

## Project Overview

This repository contains the development of the **Local Supermarket Management System for Small Shops**, created for the **CST2550 Individual Resit Coursework** at by student M00943654 at Middlesex University Mauritius.

The aim of this project is to design and implement a digital supermarket management system that replaces manual record-keeping with a more organised and efficient solution. The completed system will allow shop staff to manage products, suppliers, stock levels, sales transactions, product searches, and reports.

At this stage, the repository contains the initial planning and design documents that will be used throughout the development of the project.


## Project Objectives

The completed system will be designed to:

* Manage supermarket products and categories
* Maintain supplier information
* Monitor stock levels and low-stock alerts
* Record sales transactions
* Search products efficiently using suitable algorithms
* Generate reports to support daily supermarket operations



## Technologies

The project will be developed using the following technologies:

* C#
* WinForms
* SQL Server
* Entity Framework Core
